Web Application 2

Midterm: 8/10/2024

Finals: 29/11/2024

This course is designed to equip students with knowledge and skills to develop well-styled and responsive web applications that provide rich user experiences. COR-1702

Computational Thinking

Midterm: 9/10/2024

Finals: 30/11/2024

A course focused on teaching students the problem-solving process that involves breaking down complex problems into manageable parts, identifying patterns, and developing step-by-step solutions. It introduces key concepts such as algorithmic thinking, abstraction, decomposition, and pattern recognition. Students learn how to apply these skills to a variety of problems in different disciplines, using logic and structured approaches to create efficient solutions.
